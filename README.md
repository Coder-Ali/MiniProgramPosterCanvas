# Canvas 生成海报 - 小程序版

小程序利用 canvas 生成海报。

### Installing

```
git clone 项目名
```

## 使用方法

1.  使用微信小程序模拟器打开文件
2.  pages/index/index.html 即为主程序

## Built With

- [小程序](https://developers.weixin.qq.com/miniprogram/dev/)
- [Canvas](https://developer.mozilla.org/zh-CN/docs/Web/API/Canvas_API) - html5 特性

### 实现原理

1.  利用 JSON 配置页面数据
2.  利用小程序中的 Canvas 进行绘制
